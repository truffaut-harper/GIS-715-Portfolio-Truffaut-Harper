Mapping Lab

This lab involved the creation of two standard maps using the ArcPY language in ArcGIS. The purpose here was to introduce the class to basic functions needed to generate maps programatically.

Color Lab

This lab was used to practice the use of color within ArcGIS. The main goal was to explore different types of color contrast, resulting in two maps. The first demonstrates color harmony while the second demonstrates high contrast.

View Lab

Glyph Lab

This lab was designed to explore the use of symbology in order to communicate multiple values across multiple channels. There are four maps here, each testing a different type of symbology avalible within ArcGIS.

View Lab

Kepler Lab

This lab was designed to introduce students to Kepler.gl, an online interface which allows users to create hostable maps with time series visualizations and a 3d view.

View Lab

Small Multiples Lab

This lab introduced the concept of small multiples, meaning using several graphs with the same access to display relavent information to each other. Here, this is used to explore how different countries tweeted on two topics across time.

View Lab

Tableau Lab

This lab served as an introduction to tableau, a handy visualization tool for creating interactive data visualizations in a timely manner.
